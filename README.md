#  Python To-Do List (CLI App)

A simple command-line To-Do List application built with Python.  
It allows users to add, view, edit, and delete tasks, with data persisted using a JSON file.

This project focuses on clean object-oriented design and basic file handling.

---

##  Features

- Add new tasks
- View all tasks with numbering
- Edit existing tasks
- Delete tasks
- Persistent storage using `tasks.json`
- Simple and beginner-friendly CLI interface

---

##  Technologies Used

- Python 3
- JSON for data storage
- Object-Oriented Programming (OOP)
